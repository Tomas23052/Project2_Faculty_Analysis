1 1️⃣ QUAL ERA O PROBLEMA?

1.1 🚨 Situação Atual no IPT

  • Dados Dispersos: Informação dos docentes espalhada por múltiplas plataformas
  • Sem Visibilidade: Impossível avaliar produtividade académica e de investigação
  • Processo Manual: Avaliação de performance dependente de processos manuais
  • Decisões sem Dados: Presidência sem insights para gestão do corpo docente

1.2 🎯 Necessidades Identificadas

Centralização: - Unificar informação dos docentes - Avaliar produtividade de investigação

Automation: - Dashboard executivo - Comparação entre departamentos

Compliance: - Identificar gaps (ORCID, Scopus) - Monitorizar perfis académicos

Decision Support: - Insights data-driven - Métricas objetivas

1.3 💼 Impacto do Problema

Consequências da Situação Atual
  • Gestão Ineficiente: Recursos mal alocados
  • Oportunidades Perdidas: Talentos não identificados
  • Compliance Baixa: Perfis académicos incompletos
  • Decisões Subótimas: Baseadas em intuição, não dados

2 2️⃣ QUAL FOI A SOLUÇÃO DESENVOLVIDA?

2.1 🏗️ Arquitetura da Solução

2.2 🛠️ Tecnologias Utilizadas

Data Collection: - Web Scraping: BeautifulSoup, Selenium - PDF Processing: PyPDF2, pdfplumber, PyMuPDF - OCR: pytesseract (emails em imagem) - APIs: ORCID, scholarly (Google Scholar)

Data Processing: - pandas: Manipulação e limpeza - numpy: Operações numéricas - scikit-learn: Clustering de performance - networkx: Análise de redes

Dashboard: - Streamlit: Framework web interativo - Plotly: Visualizações avançadas

2.3 📊 Pipeline de Dados

  • Extração
  • Transformação
  • Carregamento
  • HR PDFs → Informação básica dos docentes
  • IPT “Quem é Quem” → Carga letiva, departamentos
  • ORCID → Publicações, financiamentos
  • Scopus → Métricas de citação, H-index
  • Google Scholar → Métricas adicionais
  • Limpeza e normalização de nomes
  • Matching entre fontes de dados
  • Cálculo de métricas de performance
  • Identificação de outliers e alertas
  • Datasets integrados em CSV
  • Dashboard interativo
  • Relatórios exportáveis

2.4 🔧 Funcionalidades Implementadas

Coleta Automatizada: - Scraping com rate limiting - Suporte a paginação - Tratamento de erros - Sistema de logs

Análise Avançada: - Clustering automático - Análise de redes - Benchmarking - Sistema de alertas

Dashboard Empresarial: - Interface multi-página - Filtros dinâmicos - Exportação Excel/PDF - Visualizações interativas

Robustez: - Error handling - Data validation - Quality assurance - Performance optimization

3 3️⃣ QUAIS SÃO OS BENEFÍCIOS?

Para a Gestão: - Visibilidade Total: 360° view de cada docente - Decisões Data-Driven: Métricas objetivas - Benchmarking: Comparação interna/externa - Identificação de Talentos: Top performers

Para os Docentes: - Transparência: Critérios claros - Self-Assessment: Acesso às métricas - Compliance: Identificação de gaps - Reconhecimento: Valorização
